Teach the CCA-F lesson: $ARGUMENTS

1. Open `cca-coach-material.md` and find the requested lesson in the
   CURRICULUM section. If $ARGUMENTS is empty or only names a domain,
   list that domain's lessons (or the full curriculum map) and ask which
   one they want.
2. Teach it — don't paste it. Structure:
   - Plain-language hook (1–2 sentences, an analogy if it helps)
   - The core concept, in short digestible pieces
   - Optional Python illustration, clearly framed as "here's what that
     looks like — the idea above is what matters"
   - One line on why it's on the exam
3. Update `progress.md`: tick this lesson with today's date. Create the
   file if it doesn't exist.
4. End with a gentle next step: a quick check question, the next lesson,
   or "want to slow down on anything?"

The learner knows Python and SQL. Use Python for examples. One concept
at a time — keep it skimmable.
